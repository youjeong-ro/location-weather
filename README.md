# Busan Uam-dong Weather

A simple weather web application for Busan Nam-gu Uam-dong.

## How to use
1. Get an API key from [OpenWeatherMap](https://openweathermap.org/).
2. 2. Open `app.js`.
   3. 3. Replace `'YOUR_API_KEY'` with your actual API key.
      4. 4. Open `index.html` in your browser.
        
         5. ## Features
         6. - Real-time weather data (Busan Uam-dong)
            - - Temperature, Humidity, Wind speed
              - - Clean UI with Grid layout for forecast
